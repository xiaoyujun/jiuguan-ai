# ⏳ 炼金时钟插件

一个符合"密教大会"神秘学主题的番茄钟计时器插件。

## ✨ 功能特性

### 核心功能
- ⏰ **番茄钟计时** - 默认25分钟专注时段，5分钟休息时段
- 🎨 **神秘学主题** - 完美融入项目的炼金术视觉风格
- 💬 **角色互动** - 完成后可向当前角色发送鼓励消息
- 💾 **数据持久化** - 使用localStorage保存设置和统计
- 📊 **统计功能** - 记录今日完成次数

### 界面特性
- 🌙 圆形进度环，实时显示进度
- ⚡ 流畅的动画效果
- 🎵 完成提示音（528Hz "爱的频率"）
- 📱 响应式设计，支持移动端

### 可自定义选项
- ⏱️ 自定义专注时长（1-120分钟）
- ☕ 自定义休息时长（1-60分钟）
- 💌 自定义完成消息
- 🔔 开启/关闭角色消息提醒

## 🚀 使用方法

### 启动插件

1. 在聊天输入框输入 `/`
2. 找到 "炼金时钟" 命令
3. 点击执行

或者：

```javascript
executePomodoroCommand()
```

### 基本操作

1. **开始计时** - 点击"开始"按钮
2. **暂停/继续** - 点击"暂停"按钮可暂停，再次点击"开始"恢复
3. **重置** - 点击"重置"按钮清零当前计时
4. **设置** - 点击底部"⚙️ 设置"按钮打开设置面板

### 工作流程

1. **专注阶段**（默认25分钟）
   - 开始专注工作
   - 时间到后自动提示完成
   - 如果选择了角色，将收到鼓励消息

2. **休息阶段**（默认5分钟）
   - 弹出对话框询问是否休息
   - 选择"确定"开始休息计时
   - 选择"取消"准备下一个专注时段

3. **循环重复**
   - 完成休息后，准备下一个番茄钟
   - 统计数据会自动累计

## ⚙️ 设置选项

### 专注时长
设置每个番茄钟的工作时长，范围：1-120分钟

### 休息时长
设置休息时段的时长，范围：1-60分钟

### 角色消息
- **开启**：完成时向当前选择的角色发送系统提示，角色会发送鼓励消息
- **关闭**：仅显示完成提示，不触发角色消息

### 自定义消息内容
如果开启了角色消息，可以自定义发送给角色的系统提示内容。

**默认消息模板：**
```
[系统提示] 用户刚刚完成了一个 X 分钟的专注时段（今日第 X 次），请鼓励并称赞用户的努力
```

**自定义消息示例：**
- "我刚完成了一次专注工作，需要你的鼓励和赞美"
- "完成了25分钟的学习，请夸夸我"
- "又完成一轮番茄钟，给我打打气吧"
- "专注时段结束了，用你的方式鼓励我"

角色会根据你的消息内容和她自己的性格特点做出回应。

### 统计重置
点击"重置统计"可清零今日完成次数。

## 💾 数据存储

插件使用 `localStorage` 存储以下数据：

```javascript
{
    duration: 25,              // 专注时长
    breakDuration: 5,          // 休息时长
    enableCharacterMessage: true,  // 是否发送角色消息
    customMessage: "",         // 自定义消息
    completedCount: 0          // 今日完成次数
}
```

数据存储在 `pomodoro_settings` 键中，浏览器关闭后依然保留。

### API端点

#### POST /plugins/pomodoro/complete
完成番茄钟记录

**请求参数：**
```json
{
    "role_name": "角色名",
    "duration": 25,
    "completed_count": 1
}
```

**响应：**
```json
{
    "success": true,
    "message": "专注仪式完成！持续 25 分钟",
    "timestamp": "2025-10-03 14:30:00",
    "completed_count": 1
}
```

#### POST /plugins/pomodoro/trigger-message
触发角色消息

**请求参数：**
```json
{
    "role_name": "角色名",
    "duration": 25,
    "completed_count": 1,
    "custom_message": "自定义消息（可选）"
}
```

**响应：**
```json
{
    "success": true,
    "prompt": "[系统提示] ...",
    "role_name": "角色名"
}
```

## 🎯 使用场景

1. **学习专注** - 用于学习、阅读等需要专注的活动
2. **工作计时** - 用于工作任务的时间管理
3. **习惯培养** - 配合角色鼓励，培养专注习惯
4. **休息提醒** - 防止长时间工作，定期休息
5. **统计追踪** - 了解每日专注时长

## 💡 使用建议

1. **选择角色** - 在开始前选择一个喜欢的角色，完成时会收到鼓励
2. **设定目标** - 根据任务难度调整时长，不必拘泥于25分钟
3. **避免干扰** - 专注时段关闭通知，专心完成任务
4. **适度休息** - 认真对待休息时段，让大脑放松
5. **长期坚持** - 查看统计数据，建立专注习惯
