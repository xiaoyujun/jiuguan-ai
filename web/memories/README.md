# 纪念与回顾功能模块

## 功能概述

保存和回顾重要聊天时刻的功能模块。

## 使用方法

### 方式一：聊天窗口指令（推荐）

在聊天输入框中输入 `/纪念` 或 `/纪念 数字` 指令：

- `/纪念` - 保存最近5条消息（默认）
- `/纪念 10` - 保存最近10条消息
- `/纪念 20` - 保存最近20条消息

**快捷方式**：
- 输入 `/` 打开命令索引
- 搜索"纪念"或直接选择"纪念"命令
- 按提示输入要保存的消息数量

### 方式二：直接访问页面

- 创建纪念：`/memories/page`
- 查看回顾：`/memories/review`

## 功能特性

- 📸 **快速创建**：通过聊天指令快速创建纪念
- 🖼️ **图片支持**：上传图片记录时刻（可选）
- 📝 **描述文字**：添加回忆描述
- 🔍 **智能搜索**：按关键词和角色搜索
- 🎬 **场景还原**：将纪念还原到当前对话

## 数据存储

所有纪念数据存储在：`聊天记录/纪念/`

## 技术实现

- 后端：Flask蓝图 (`web/memories/`)
- 前端：HTML + CSS + JavaScript
- 集成：命令索引系统 (`command_index.js`)
