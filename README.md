# 酒馆 AI

基于 Flask 的 AI 角色扮演/聊天应用，前端使用原生 HTML/CSS/JavaScript，数据主要采用 JSON/YAML 文件存储。

## 快速开始

1. 安装依赖
   - Windows: `安装依赖.bat`
   - 或命令行: `python -m pip install -r requirements.txt`
2. 启动项目
   - Windows: `启动.bat`
   - 或命令行: `python web/app_new.py`
3. 访问地址
   - 默认: `http://127.0.0.1:8000`

## 目录说明

- `web/`: Flask 主应用与前端静态资源
- `API.py`: 模型调用和路由封装
- `data/`: 项目数据目录
- `doc/`: 文档目录
